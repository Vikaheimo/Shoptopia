### Frontend

Frontend uses Next with Tailwind and Typescript.

### Backend

Backend uses Express with Typescript and MongoDB as it's database.

| Requirement | Version tested |
|-|-|
| Node | 18.16.1 |
| Express | 4.18.2 |
| MongoDB | 6.0.3 |
| Typescript | 5.0.2 |
